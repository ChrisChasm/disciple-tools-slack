== Description ==

[youtube https://www.youtube.com/watch?v=iIdapuXsZuY]

Ultimate Slack Notifications is a robust solution to notify team members with all selected activities happening inside the WordPress websites that need prime attention.  The plugin posts all notifications inside the admin defined **Slack channels** and simultaneously shows **Desktop Notifications** just after when events are fired in WordPress sites. 

Ultimate Slack Notifications is easy to use, allows to post specific notifications in particular Slack Channels so that each task can easily be assigned to eligible team members.  

Ultimate Slack Notifications is **highly customizable and extendable**, has been developed to pull notifications based on the **WordPress core events and hooks**. So, anyone having lest understanding of WordPress architecture can easily maximize the plugin features easily. 



 **Plugins Features**
 
* Create five Individual setting to post customized notification in Slack Channels.
* Unlimited Desktop Notification  
* Easy to use and Infinitely Customizable
* Powerful and sleek Admin Interface
* Create notification for 3rd party plugin events like Buddypress, bbPress 
* Send test/demo into associated Slack Channels creating desktop notifications.


**List of notifications for WordPress core activities**

* When a post is created
* When a post is published
* When a post is updated
* When a post is trashed
* New User Registration
* New Post and page comments

**Notification List of 3yrd Party WordPress Plugins**

* Buddypress: New Status
* bbPress: New Topic

**Developer Oriented Features**

The Slack notification plugin is powerful and efficiently programmed, especially crafted for developers. The plugin builds connection between Slack and WordPress using Slack provided **Incoming Webhooks Bot**  to post WordPress site data into Slack channels real-time.  Ultimate Slack Notifications calls main methods **“ppwp_post_to_slack”** to post inside the Slack which can be called from anywhere in the WordPress. If you need learn about the parameters, open **ppwpslack.php** from **ppwpslack** folder and you will find the parameters residing in ppwp_post_to_slack method. 

**Language Support**

* British English

== Installation ==

[youtube https://www.youtube.com/watch?v=iIdapuXsZuY]

Installing Unlimited Slack Notification plugin is the same as other WordPress plugins. On top of that, I have chunked down the installation and usage process. 

**To install the plugin:**

1. Install the plugin through Plugins > Add New > Upload or unzip plugin package into wp-content/plugins/.
02. Activate the Ultimate Slack Notifications through the 'Plugins > Installed Plugins' menu in WordPress.
